Data:
This is meant for prefabs and other type of data files.

H-A Assets:
All project assets go here, any models, animations, UI images, Audio, etc.
(Both Self made and imported, theres a specific folder for any imported assets in this folder.)

Other:
Any kind of files that just clutter up the main asset folder and dont have a specified place.

Scenes:
For Scenes and scene info.

Scripts:
All scripts for anything.

Tools-Animation:
Animation tools, if adding other type of tools, add a folder called "Tools" and put "Tools-Animation in it".
